bilang-slidekeys — userscript demo

This page simulates what you'd see on any web page after the Tampermonkey userscript loads. The floating keyboard button should appear in the bottom-right corner.

Try it:

  1. Click the input below
  2. Click the floating ⌨️ button to summon the keyboard
  3. Or press Ctrl+Shift+K

Excluded sites won't get the keyboard. Everywhere else, it's available.
