bilang-slidekeys

A bilingual on-screen keyboard for English / Ukrainian (and German, Spanish, French, Italian). Live demos below.

github.com/grogzoid/bilang-slidekeys ↗

LiveType in action — typing on a physical QWERTY keyboard produces Ukrainian via the standard ЙЦУКЕН mapping.

Interactive demos

Playground playground.html

Mode switcher, Latin language picker, plain inputs and textareas, plus an event log. The kitchen-sink page for poking at every option.

 A1 Translation Quiz quiz.html

Translate 10 English phrases into Ukrainian. Submit-all with optional corrections. Practical learner format.

 Bilingual Chat chat.html

Two-pane chat console: read English, reply in Ukrainian. Shows the keyboard in a real-feeling messaging UI.

 Shevchenko Typewriter poetry.html

Type "Заповіт" character by character. Ghost text guides you, correct keys glow green, mistakes red. Letters and accuracy at the end.

 Borscht Recipe Card recipe.html

Build a bilingual recipe card with side-by-side EN / UK fields. Practical content-creation use case.

 Showreel auto-play showreel.html

Self-playing animated demo, loops indefinitely. Useful as a screen recording source or marketing material.
